# Roomoria - Modern Hotel Booking Website

Roomoria adalah website pemesanan hotel modern yang dibangun menggunakan **React + TypeScript + TailwindCSS**.  
Project ini dirancang dengan tampilan clean, elegan, dan responsif, serta menyediakan fitur lengkap mulai dari hero section, daftar kamar, detail kamar, form booking, hingga booking summary.

## Features

### **Landing Page**

-   Hero section dengan tampilan modern dan elegan.
-   Section “Our Rooms” menampilkan daftar kamar lengkap dengan gambar, harga, dan detail.
-   Section “About” untuk informasi hotel.
-   Section “Fasilitas” menampilkan fasilitas unggulan hotel.
-   Footer profesional yang responsif.

### **Room Booking System**

-   Detail room page dengan informasi kamar lengkap.
-   Form booking dengan input:
    -   Nama lengkap
    -   Tanggal check-in
    -   Tanggal check-out
    -   Jenis kamar
    -   Total tamu
    -   Catatan tambahan
-   Halaman **Booking Summary** yang menampilkan ringkasan pemesanan.

### **Navigation**

-   Navbar terhubung dengan:
    -   Home → tetap di halaman utama
    -   Rooms → scroll ke section “Our Rooms”
    -   Booking → masuk halaman booking
    -   About → scroll ke section “About”

## Tech Stack

| Teknologi            | Fungsi                            |
| -------------------- | --------------------------------- |
| **React.js**         | Library UI utama                  |
| **TypeScript**       | Supaya lebih aman dan terstruktur |
| **React Router DOM** | Routing antar halaman             |
| **TailwindCSS**      | Styling cepat, modern, responsif  |
| **Lucide Icons**     | Icon premium dan clean            |

## Cara Menjalankan

### 1. Clone repository

```sh
git clone https://github.com/rizkicahyaa/roomoria.git
cd roomoria
```

### 2. Menginstall npm

```sh
npm install
```

### 3. Jalankan aplikasi

```sh
npm run dev
```
